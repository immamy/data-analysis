# Google News Word Cloud Scraper
This project crawls News from Google and generates word cloud visualization based on word frequency analysis.

Technology: **Python, requests, pandas, jieba, Matplotlib**

## File
 `side-project-wordcloud.py`  

## Output
 `side-project-wordcloud_word-frequency.csv`  
 `side-project-wordcloud_word-cloud-plt.png`  
 `side-project-wordcloud_Bar-chart-plt.jpg`  
 `side-project-wordcloud_Pie-chart-plt.jpg`  

## Features
- Crawls news data from Google News.
- Performs word frequency analysis.
- Implemented data visualization using matplotlib.

## Workflow
1.Crawl Google News.
2.Remove punctuation.
3.Remove stopwords.
4.Chinese word segmentation.
5.Word frequency calculation.
6.Generate word-cloud-plt, Bar-chart-plt, Pie-chart-plt.
