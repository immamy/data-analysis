# PTT Comment Word Cloud Scraper
This project crawls comments from PTT posts and generates word cloud visualization based on public discussions.

Technology: **Python, requests, pandas, jieba, Matplotlib**

## File
 `side-project-ptt.py`  
 `side-project-ptt-wordcloud.py`  

## Output
- Word frequency data
- Word cloud image
- Bar chart image
- Pie chart image

## Features
- Crawl PTT post comments.
- Clean text data.
- Chinese word segmentation using jieba.
- Word frequency analysis.
- Generate word cloud visualization.

## Workflow
1.Crawl PTT comments.

2.Merge all comments into one text file.

3.Remove punctuation.

4.Remove stopwords.

5.Chinese word segmentation.

6.Word frequency calculation.

7.Generate word cloud.
